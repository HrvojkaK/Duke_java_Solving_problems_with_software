
A simple program that takes in multiple color images (prompts for selection), turns them into black and white ones, and saves B&W copies, with names "BW_" + original image name.

Uses edu.duke library - this comes with Duke Uni's edition of BlueJ environment, downloadable from
https://www.dukelearntoprogram.com//course2/index.php
The library can also be downloaded separately as JAR, from:
https://www.dukelearntoprogram.com//downloads/bluej.php?course=2
(edu.duke link, bottom of the page).



**assignment 4, for Duke Uni's Java programming: Solving Problems with Software course on Coursera**
